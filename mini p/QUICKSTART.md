# Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Run the Application
```bash
python app.py
```

### Step 3: Open in Browser
Navigate to: **http://localhost:5000**

## 📝 What You Can Do

1. **Upload Your Resume** (PDF, DOCX, or TXT)
2. **Get Instant AI Analysis** including:
   - ATS Compatibility Score
   - Skills Detection
   - Contact Information Extraction
   - Experience Analysis
   - Personalized Suggestions

## 🎯 Key Features

✅ **AI-Powered Analysis** - Advanced resume parsing and analysis  
✅ **ATS Score** - Know how well your resume will pass ATS systems  
✅ **Smart Suggestions** - Get actionable tips to improve your resume  
✅ **Skills Detection** - Automatically identifies your technical skills  
✅ **Beautiful UI** - Modern, responsive design  

## 💡 Tips

- Upload a resume with selectable text (not just images)
- PDF files work best
- The analysis is instant and free!
